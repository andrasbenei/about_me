Sudoku

With this script, the User can play Sudoku or let the computer solve the table. It provides the User a board with 20 given numbers. The User can place numbers on the board according to the rules of Sudoku. The User can also make the script solve the board with the backtracking method. In this case, the script shows the steps with some delay so the User can follow them.
